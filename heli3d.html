SETUP — GOOGLE MAPS API KEY

写真リアルな3D都市(Google Photorealistic 3D Tiles)の表示には、あなた自身の Google Maps Platform APIキーが必要です。キーはこのブラウザ内(localStorage)にのみ保存され、外部に送信されるのはGoogleへの地図リクエストだけです。

  1. Google Cloud Console でプロジェクトを作成(要アカウント・無料枠あり)
  2. 「APIとサービス → ライブラリ」で Map Tiles API を有効化
  3. 「認証情報」で APIキー を作成してコピー
